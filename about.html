About This Project

In April 2025, the RBI mandated all Indian banks to migrate to .bank.in domains by October 31, 2025, to reduce phishing and spoofing. The registry is operated by IDRBT.

This scorecard provides an independent, automated security rating for every registered .bank.in domain. It is built and maintained by CashlessConsumer as a public transparency initiative.

Data is refreshed daily via automated scans. Last scan: .

Scoring Rubric (0-100)

HTTPS/TLS (15 pts)
10 for HTTPS, +5 for TLS 1.3, +3 for TLS 1.2
HSTS (10 pts)
7 for HSTS header, +3 for preload
Security Headers (15 pts)
5 each for X-Frame-Options, X-Content-Type-Options: nosniff, CSP
Email Security (20 pts)
12 for DMARC reject, 8 for quarantine, 4 for any DMARC. +4 for SPF, +4 for MX
DNSSEC (10 pts)
10 if DNSKEY record present
No Bad Exposures (20 pts)
5 each: no .git leak, no .env leak, no phpinfo, no cPanel on standard ports
Site Reachable (10 pts)
5 if returns 200/301/302, +5 if not timeouting

Grade Boundaries

A ≥ 80 Excellent
B ≥ 60 Good
C ≥ 40 Average
D ≥ 20 Poor
F < 20 Critical
